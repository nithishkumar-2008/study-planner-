🔧 One-time Setup needed for Google Sync

Oru nimisham Google Cloud la setup pannanum (free, one-time only):
1. Go to console.cloud.google.com → Create a new project
2. Search "Google Drive API" → Enable it
3. Go to APIs & Services → OAuth consent screen → External → Fill name → Save
4. Go to Credentials → Create Credentials → OAuth 2.0 Client ID
5. Application type: Web application
6. Authorized JavaScript origins: add
7. Copy the Client ID and paste below 👇
⚠️ Client ID is stored only in your browser (localStorage). Never shared.
